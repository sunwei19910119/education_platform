使用Python3.x与Django2.0.1开发的在线教育平台网站

## Quick Start

```
$ git clone https://git@github.com:sunwei19910119/education_platform.git
$ cd Mxonline3
$ pip install -r requirements.txt
$ python manage.py runserver
```

use the address: http://127.0.0.1:8000/
